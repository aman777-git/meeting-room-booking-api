# Meeting Room Booking API

Spring Boot REST API to manage meeting rooms and bookings with time-slot conflict detection.

## Tech Stack
- Java 17+
- Spring Boot
- Spring Data JPA
- Hibernate
- H2 / PostgreSQL
- Maven
- Lombok

## Features
- Create & list meeting rooms
- Book rooms with conflict validation
- DTOs + validation
- Service + ServiceImpl architecture
- Global exception handling

## Run Project
```bash
mvn spring-boot:run
